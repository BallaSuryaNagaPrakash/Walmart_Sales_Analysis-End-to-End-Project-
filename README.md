# Walmart_Sales_Analysis-End-to-End-Project-
This project analyzes Walmart sales data to uncover key business insights using Power BI. The dashboard provides an interactive visualization of sales performance across branches, cities, product categories, and customer behavior.

------------------------------------------------------------------------------------------------------------------------------------------------------

📌 Project Overview

The Walmart Sales Analysis Dashboard is a data analytics project that analyzes Walmart retail sales data to identify key business insights. Using interactive dashboards in Power BI, this project helps understand sales trends, customer behavior, product performance, and branch-wise revenue.

The goal of the project is to transform raw sales data into meaningful insights that support data-driven business decisions.

🎯 Business Problem

Retail companies generate large volumes of sales data every day. Without proper analysis, it becomes difficult to identify:

Which branches generate the highest revenue

Which product categories perform best

Customer purchasing behavior

Preferred payment methods

Sales trends over time

This project solves these challenges by building an interactive Power BI dashboard that allows users to easily explore and analyze sales performance.

🎯 Project Objectives

Analyze Walmart sales performance across branches and cities

Identify top-performing product categories

Understand customer purchase behavior

Analyze payment method trends

Track monthly and daily sales trends

Build an interactive dashboard for business insights

📂 Dataset Information

The dataset contains Walmart transactional sales data with multiple attributes describing sales performance and customer activity.

Dataset Columns
Column	Description
Invoice ID	Unique transaction identifier
Branch	Store branch location
City	City where the store operates
Customer Type	Member or Normal customer
Gender	Gender of the customer
Product Line	Category of the product
Unit Price	Price per product
Quantity	Number of products purchased
Tax	Tax applied to the purchase
Total	Total transaction amount
Date	Date of purchase
Time	Time of purchase
Payment	Payment method used
Rating	Customer satisfaction rating

📊 Dashboard Features

The Power BI dashboard includes multiple interactive visualizations such as:

Total Sales KPI

Total Quantity Sold

Average Customer Rating

Sales by City

Sales by Payment Method

Branch-wise Sales Performance

Monthly Sales Trend

Users can interact with the dashboard using filters and slicers to explore the data dynamically.

📈 Key Insights

Some key insights discovered from the analysis include:

Certain product lines generate higher revenue compared to others.

Some store branches perform better in overall sales.

Customers show clear preferences for specific payment methods.

Sales fluctuate across different time periods.

These insights can help businesses improve sales strategies and customer experience.
